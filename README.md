# 🌍 InAmigos Foundation Awareness Webpage

A responsive awareness webpage developed using **HTML5** and **CSS3** as part of my **Web Development Internship** at **InAmigos Foundation**.

The objective of this project was to design a clean, informative, and visually engaging webpage that spreads awareness about the foundation's mission, ongoing initiatives, and the positive social impact created through its various community projects.

---

## 📖 About the Project

This webpage introduces visitors to **InAmigos Foundation**, a Section 8 registered non-profit organization dedicated to creating meaningful social change.

The website highlights the foundation's vision, ongoing projects, impact on society, and encourages visitors to become volunteers or support the organization's initiatives.

The design focuses on simplicity, readability, and accessibility while maintaining a modern user interface.

---

## ✨ Features

- Responsive and clean webpage layout
- Hero section introducing the NGO
- About InAmigos Foundation section
- Details of six ongoing NGO projects
- Social impact section
- Call-to-action encouraging volunteering and support
- Social media links for easy connectivity
- Modern card-based project layout
- Fully built using HTML and CSS

---

## 🌱 NGO Projects Featured

- 🍚 **Seva** – Providing food, clothes, and essential support to those in need.
- 📚 **Bachpanshala** – Supporting the education and care of underprivileged children.
- 🐾 **Jeev** – Animal rescue, care, and welfare initiatives.
- 👩‍🎓 **Udaan** – Empowering women through education, skills, and opportunities.
- 🌿 **Prakriti** – Environmental conservation through plantation drives and cleanliness campaigns.
- 🚀 **Vikas** – Youth development through internships, mentorship, and skill-building programs.
